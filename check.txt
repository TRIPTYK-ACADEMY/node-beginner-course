Exercice 1
----------

Vous définissez une fonction qui prend 2 paramètres : 1 string, et 1 fonction dite "callback".
Le corps de cette fonction doit effectuer les traitements suivants :
- Retourner la string. Par exemple, "Hello" devient "olleH".
- Afficher la string traitée en console via la fonction de callback.
Vous exécutez cette fonction pour démontrer le comportement attendu.

Exercice 2
----------

Vous définissez une fonction qui prend un paramètre qui sera un entier.
Cette fonction retourne une promesse qui doit être rejetée si le paramètre n'est pas un entier, et qui doit être résolue si le paramètre est bien un entier. 
Le rejet retourne un message d'erreur, la résolution retourne l'entier multiplié par 10.
Vous exécutez cette fonction pour démontrer le comportement attendu.
