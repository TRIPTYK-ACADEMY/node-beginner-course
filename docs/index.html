1. Introduction

Qu'est-ce que c'est ?

Environnement d'exécution Javascript construit sur le moteur JavaScript V8 de Chrome.
C'est un environnement bas niveau permettant l’exécution de JavaScript côté serveur.

Avantages

Node.js utilise un modèle basé sur l’événementiel et des entrées/sorties non bloquantes, ce qui le rend léger et efficace.
C'est ultra-rapide !
L’écosystème de logiciels de Node.js, npm, est le plus grand écosystème de bibliothèques open source au monde !
Les coûts de production sont très bas par rapport aux technologies concurrentes.

Inconvénients

Fragilité du langage Javascript et de la programmation fonctionnelle vs la programmation orientée objet
Grosse exposition aux hackers
Le bon côtoie le moins bon

Historique

Node.js a été créé par Ryan Dahl en 2009, qui a eu cette idée après avoir observé la barre de progression d'un chargement de fichier sous Flickr : le navigateur ne savait pas quel pourcentage du fichier était chargé, et devait adresser une requête au serveur web. Dahl voulait développer une méthode plus simple et quand le moteur V8 fut diffusé, il commença à s'intéresser à JavaScript : node.js était né !

Wikipédia - https://fr.wikipedia.org/wiki/Node.js

2. Utilisation en ligne de commande

Un premier script node.js

  • Créer un répertoire Exo-01 à la racine du cours.
  • Créer un fichier javascript qui va contenir le code à exécuter.
  • Exécuter ce fichier et afficher un bel "Hello Technocité" dans le terminal.
  • Level up : utiliser une chaîne de caractères "Hello World" qui, à l'affichage, sortira en "Hello Technocité".
  • Level up up : afficher une chaîne de caractères "Hello World" pendant 2s, puis afficher "Sorry, the good string would be displayed in 3 seconds ...", et enfin afficher "Hello Technocité"

                      console.log('Hello World !');
                    

                      console.log('Hello World !'.replace('World', 'Technocité'));
                    

                    console.log('Hello World !');
                    setTimeout(function() {
                      console.log('Sorry, the good string would be displayed in 3 seconds ...');
                    }, 2000);
                    setTimeout(function() {
                      console.log('Hello Technocité');
                    }, 5000);
